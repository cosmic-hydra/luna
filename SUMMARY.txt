INSTALLATION PROCESS
================================================================================

1. Install Termux apps from F-Droid:
   - Termux
   - Termux:Boot
   - Termux:API

2. Clone repository:
   git clone https://github.com/cosmic-hydra/luna.git
   cd luna

3. Run installer:
   ./install-termux.sh

4. Configure:
   nano ~/.openclaw/openclaw.json
   (Add API keys, set password)

5. Start service:
   ~/openclaw-start.sh

6. Access web interface:
   http://YOUR_IP:18789

Total time: ~10-15 minutes (depending on internet speed)
